Golf Card Game

This was a project I created while working in a group for my AI course in the Spring of 2025. I was tasked with implementing the Golf card game into Python, levering Pygame for the graphical interface. The project was marked for distinction by the professor.

Project Overview

Purpose: To create a sustainability themes implementation of the classic card game 'Golf' with both AI and human opponents.

My Work: Built a complete card game using Python and Pygame for the graphical interface. Implemented both Player vs. Player and Player vs. AI game modes with three difficulty levels. The AI uses the Minmax Algorithm with Alpha-Beta pruning and custom heuristics to make optimal strategic moves.

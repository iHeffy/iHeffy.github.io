University of St. Thomas Database Intern

Purpose: To enhance the Database I course curriculum by integrating a relational algebra Python interpreter into the EdStem learning platform, as the previous relation algebra assignments were to be completed on paper and hand graded.

My Work: Integrated a Python-based relational algebra interpreter into an EdStem coursethat allows students to practice and visualize database via relational algebra operators. Trash Point

Purpose: To develop a comprehensive waste management platform that monitors trash bin capacity in real-time to optimize collection routes and reduce environmental impact.

My Work: Collaborated with two peers to build a full-stack application using React for the frontend, Node.js for the backend, and Supabase with PostgreSQL for the database. Implemented real-time monitoring capabilities and utilized Leaflet for interactive mapping. Created comprehensive database design using Lucid Chart with crow's foot diagrams.

Technologies: React, Node.js, Supabase, PostgreSQL, Leaflet, Lucid Chart

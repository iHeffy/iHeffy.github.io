RADB into EdStem

This past Summer I got to work with Dr. Hoefer as a Database intern to work on a project to integrate a Relational Algebra interpreter into the EdStem course. This would effectively automate the first three assignments cover RADB as well as allowing for students to check their work on assignments in hopes to improve the learning of the material.

Project Overview

Purpose: To enhance the Database I course curriculum by integrating a relational algebra Python interpreter into the EdStem learning platform, as the previous relation algebra assignments were to be completed on paper and hand graded.

My Work: Integrated a Python-based relational algebra interpreter into an EdStem coursethat allows students to practice and visualize database via relational algebra operators. This tool helps future Database I students better understand relational algebra concepts and their connection to SQL queries, and allows real time feedback on their answers.

def grade():
    output_lines, error = run_radb_query(DB_PATH, STUDENT_FILE)
    test_cases = []

    if error:
        test_cases.append({
            "name": "RADB execution",
            "score": 0,
            "max_score": 1,
            "ok": False,
            "feedback": f"Error running RADB: {error}",
            "hidden": False,
            "private": False,
        })
    else:
        cleaned_output = clean_radb_output(output_lines)
        expected_set = set(expected_output_lines)
        actual_set = set([line.strip() for line in cleaned_output])

        missing = expected_set - actual_set
        extra = actual_set - expected_set

        test_cases.append({
            "name": "Match unordered output",
            "score": 1 if not missing and not extra else 0,
            "max_score": 1,
            "ok": not missing and not extra,
            "feedback": f"Missing: {missing}, Extra: {extra}" if missing or extra else "",
            "hidden": False,
            "private": False,
        })

    print(json.dumps({"testcases": test_cases}, indent=2))

# run the grader
grade()

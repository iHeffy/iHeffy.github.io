Golf Card Game

Purpose: To create a sustainability themed implementation of the classic card game 'Golf' with both AI and human opponents.

My Work: Built a complete card game using Python and Pygame for the graphical interface. Implemented both Player vs. Player and Player vs. AI game modes with three difficulty levels. The AI uses the Minmax Algorithm with Alpha-Beta pruning and custom heuristics to make optimal strategic moves.

Technologies: Python, Pygame, Minmax Algorithm, Alpha-Beta Pruning

Project Demo

Project Details

This project demonstrates advanced algorithmic thinking and game development skills. The implementation includes:

  • Complete card game logic with proper shuffling and dealing
  • AI opponent using Minmax algorithm with Alpha-Beta pruning for optimal performance
  • Three difficulty levels (Easy, Medium, Hard) with different AI strategies
  • Pygame-based graphical interface with intuitive controls
  • Sustainability theme integration throughout the game design

The AI implementation showcases understanding of game theory and optimization algorithms, making strategic decisions based on current game state and potential future outcomes.
